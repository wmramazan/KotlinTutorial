# KotlinTutorial
A tutorial application which uses Kotlin language and some Jetpack components.

## Used Jetpack Components
- ViewModel
- LiveData
- Room
- Data Binding

## Features
- Lists and caches data gets from the server.
- Shows the error message which returns from the server.
